HS-Brexit ACTOR Supervised Model
================================

Dataset:
HS-Brexit

Model:
ACTOR (Active Learning with Annotator-specific Classification Heads)
Shared bert-base-uncased encoder with annotator-specific classification heads

Architecture:
- Shared BERT encoder
- 6 annotator-specific linear heads
- CLS embedding used as sentence representation
- Each example routed through the head corresponding to its annotator

Training setup:
epochs: 3
batch size: 8
learning rate: 2e-5
weight decay: 0.01
evaluation metric for model selection: macro_f1

Dataset sizes (annotation-level):
train: 4704
dev: 1008
test: 1008

Annotators:
Ann1, Ann2, Ann3, Ann4, Ann5, Ann6

Train label distribution:
0 (non-hate): 4099
1 (hate): 605

Test results:
eval_loss: 0.3660
eval_accuracy: 0.8889
eval_macro_f1: 0.7370
eval_weighted_f1: 0.8845

Notes:
This experiment implements the supervised ACTOR architecture without the active
learning loop.

The model uses one shared BERT encoder and one classification head per
annotator. Each training instance is routed through the head corresponding
to the annotator who produced that label.

This result is compared against:
1. Majority-vote BERT baseline
2. Single-annotation BERT baseline
3. ACTOR supervised model
